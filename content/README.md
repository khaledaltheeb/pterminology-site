# نظام توسيع محتوى «مصطلحات علم النفس»

هذا المجلد هو نقطة الإدخال المستقبلية للمحتوى، بحيث لا يلزم تعديل القوالب أو GitHub Actions عند إضافة دفعات جديدة.

## صيغة CSV المعتمدة

الأعمدة:

```text
slug,ar,en,category,description,keywords,related,status,reviewed_at
```

القواعد:

- `slug`: معرّف لاتيني فريد وثابت، بحروف صغيرة وشرطات فقط.
- `ar`: الاسم العربي الدقيق.
- `en`: الاسم الإنجليزي أو المقابل العلمي.
- `category`: تصنيف واحد واضح.
- `description`: شرح أصلي مفيد لا يقل عن 120 حرفًا.
- `keywords`: كلمات مرتبطة مفصولة بعلامة `|`.
- `related`: slugs لموضوعات مرتبطة، مفصولة بعلامة `|`.
- `status`: `draft` أو `reviewed` أو `published`.
- `reviewed_at`: تاريخ المراجعة بصيغة YYYY-MM-DD.

## سير العمل

1. انسخ `import-template.csv` إلى ملف دفعة جديد.
2. أضف المصطلحات دون تكرار.
3. اجعل الحالة `reviewed` بعد المراجعة.
4. ارفع الملف إلى هذا المجلد.
5. يبني النظام الصفحات، JSON، CSV، الروابط الداخلية وخرائط الموقع تلقائيًا.

## اختبارات النشر

يفشل النشر تلقائيًا عند:

- تكرار slug أو العنوان العربي.
- نقص العنوان أو الوصف.
- عدم تطابق عدد الصفحات مع البيانات.
- وجود Sitemap غير صالح.
- اختفاء ملف تحقق Google.
- ظهور روابط داخلية مكسورة في الاختبارات.

## الوضع الحالي

- 2000 مفهوم/موضوع نفسي مستقل.
- 200 صفحة مركزية.
- Sitemap Index مقسّم إلى خرائط صغيرة.
- ملفات API وتنزيل CSV.
