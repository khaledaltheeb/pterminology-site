# عقد المصادر والأدلة العلمية v75

## الهدف

يوحد هذا العقد طريقة تسجيل المصادر وربطها بالادعاءات في محتوى «مصطلحات علم النفس». وجود رابط في نهاية الصفحة لا يكفي وحده لإثبات أن كل جملة عالية المخاطر مدعومة؛ المطلوب لاحقًا معرفة المصدر، نوعه، تاريخ التحقق منه، حالته، والادعاءات المحددة التي يسندها.

## الوضع الانتقالي

المستودع يحتوي حاليًا على سجلات مصادر أساسية بصيغة `publisher`, `title`, `url`, `year`. لا تُفشل هذه السجلات القديمة لمجرد أنها لم تُرحّل بعد؛ يصنفها المدقق كتحذيرات ويقيس حجم العمل المتبقي.

تُفشل البوابة فورًا عند وجود:

- مصدر ليس كائن JSON صالحًا.
- نقص في الحقول الأساسية.
- رابط غير HTTPS أو غير مطلق.
- سنة غير رقمية أو غير معقولة.
- تكرار الرابط نفسه داخل قائمة المصادر الواحدة.
- بدء إضافة البيانات المتقدمة ثم ترك جزء منها ناقصًا.

## السجل الكامل المطلوب

عند إنشاء محتوى جديد أو مراجعة محتوى عالي الخطورة، يضاف إلى كل مصدر:

- `id`: معرف ثابت بصيغة lowercase kebab-case.
- `publisher`: الجهة الناشرة.
- `title`: عنوان المصدر.
- `url`: رابط HTTPS مباشر.
- `year`: سنة النشر أو آخر تحديث المعلن.
- `source_type`: نوع المصدر المحدد في العقد.
- `verified_at`: تاريخ التحقق من الرابط والمحتوى بصيغة YYYY-MM-DD.
- `claims_supported`: قائمة بالادعاءات أو المحاور التي يسندها المصدر.
- `status`: حالة المصدر الحالية.

## مستويات المخاطر

### منخفض

لغة الواجهة والتنقل والنصوص غير السريرية. لا تحتاج دائمًا إلى مرجع خارجي.

### متوسط

التثقيف العام والعلامات الشائعة والأثر الوظيفي. يتطلب مصدرًا مؤسسيًا أو مراجعة موثوقة على الأقل.

### مرتفع

التشخيص والتشخيص التفريقي والعلاج والتثقيف الدوائي وحماية الأطفال وذوي الإعاقة. يتطلب مصدرين على الأقل، مع إرشاد أو تصنيف أو مراجعة منهجية مناسبة.

### حرج

إيذاء النفس والعنف والطوارئ وقرارات الدواء وادعاءات صلاحية المقاييس. يتطلب تحققًا أقرب ومراجعة بشرية متخصصة، ولا يجوز تمريره آليًا باعتباره مجرد محتوى تثقيفي.

## حالات المصادر

- `current`: صالح للاستخدام الحالي ضمن نطاقه.
- `superseded`: استُبدل بإصدار أحدث.
- `withdrawn`: سحبته الجهة أو المجلة.
- `inaccessible`: تعذر الوصول إليه ويحتاج بديلًا أو تحققًا.
- `pending_verification`: أضيف لكنه غير مصرح بالاعتماد عليه بعد.

المصدر المنتهي أو المسحوب لا يُحذف من الأثر التدقيقي، لكنه لا يستخدم لتسويغ ادعاء حالي دون توضيح تاريخي صريح.

## مخرجات المدقق

ينتج `artifacts/source-evidence-v75.json` ويشمل:

- عدد الملفات التي تحتوي قوائم مصادر.
- إجمالي سجلات المصادر.
- عدد السجلات الجاهزة للعقد.
- عدد السجلات القديمة المطلوب ترحيلها.
- الأخطاء التي تمنع الدمج.
- التحذيرات وقائمة السجلات ومواقعها داخل JSON.
