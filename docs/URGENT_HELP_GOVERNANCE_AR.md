# حوكمة مسارات المساعدة العاجلة

## الغرض

تمنع هذه السياسة نشر أرقام طوارئ أو جهات مساعدة نفسية غير موثقة أو قديمة أو غير مناسبة لبلد المستخدم. المنصة تثقيفية؛ لا تشخّص الأزمات، ولا تقدر مستوى الخطر الفردي، ولا تستبدل خدمات الطوارئ أو التقييم المهني العاجل.

## القاعدة الافتراضية

كل رقم أو خدمة أو جهة **ممنوعة من النشر افتراضيًا**. لا يكفي ظهور الرقم في نتائج البحث أو منشور اجتماعي أو دليل تجاري. لا يسمح بالنشر إلا لسجل بحالة `verified` يستوفي عقد البيانات كاملًا ويظل داخل مدة المراجعة.

## الأولوية عند الخطر الوشيك

عند وجود خطر وشيك على النفس أو الآخرين، يسبق الإجراء الآمن أي شرح أو اختبار. يجب أن يرى المستخدم مسار المساعدة دون إنشاء حساب أو إكمال نموذج أو انتظار نتيجة. يكون التوجيه العام المحدود إلى خدمات الطوارئ المحلية أو أقرب قسم طوارئ، مع الاستعانة بشخص موثوق والبقاء مع الشخص عند القدرة الآمنة.

## متطلبات إدراج أي خدمة

لا تنشر خدمة إلا بعد توثيق:

1. البلد والولاية القضائية التي تخدمها.
2. الاسم الرسمي ونوع الخدمة.
3. وسيلة الاتصال ونوعها كما تظهر في المصدر الرسمي.
4. رابطًا رسميًا مباشرًا وسلطة المصدر.
5. تاريخ التحقق وتاريخ المراجعة التالية وطريقة التحقق.
6. ساعات التوفر واللغات والفئات المستهدفة.
7. التكلفة والسرية كما يؤكدهما المصدر، أو التصريح بعدم المعرفة.
8. حدود النطاق وأي معلومات غير مؤكدة.
9. الأسطح المسموح بالنشر عليها.

مدة المراجعة القصوى في العقد 90 يومًا، مع إعادة تحقق فورية عند تغير الرقم أو وسيلة الاتصال أو الجهة المشغلة أو النطاق أو ساعات العمل. يتطلب تغيير وسيلة الاتصال فحصًا ثانيًا مستقلًا قبل النشر.

## المصادر المقبولة والمرفوضة

تقبل المصادر الحكومية، وخدمات الصحة الوطنية، وخدمات الطوارئ الرسمية، والجهات المعيّنة رسميًا لتقديم دعم الأزمات. لا تكفي مقتطفات البحث أو منشورات التواصل الاجتماعي أو تعليقات المستخدمين أو الأدلة غير الموثقة أو إجابة مولدة بالذكاء الاصطناعي.

## ما يجب إظهاره للمستخدم

يظهر بجوار كل خدمة منشورة:

- البلد والنطاق الجغرافي.
- تاريخ التحقق وتاريخ المراجعة التالية.
- ساعات التوفر واللغات والفئة المستهدفة.
- ملاحظة التكلفة والسرية وحدودهما.
- أي عدم يقين أو قيد معلوم.

لا توصف الخدمة بأنها سرية أو متاحة 24/7 أو مجانية ما لم يذكر المصدر الرسمي ذلك صراحة وبالنطاق نفسه.

## ما يمنع ادعاؤه

يمنع وصف جهة بأنها معتمدة من المنصة أو الأفضل، أو الادعاء بأن الاتصال يضمن السلامة، أو أن الخدمة تعمل عالميًا، أو أن جميع الاتصالات سرية، أو أن الخدمة متاحة دائمًا أو مجانية للجميع دون دليل رسمي. كما يمنع الادعاء بوجود مراجعة اختصاصي أو شراكة لم تحدث.

## المسار الآمن عند غياب خدمة محلية موثقة

عندما لا تتوفر خدمة محلية مؤكدة، لا يخترع الموقع رقمًا عالميًا. يعرض فقط توجيهًا عامًا محدودًا:

- عند الخطر الوشيك: طلب خدمات الطوارئ المحلية أو التوجه إلى أقرب قسم طوارئ.
- البقاء مع الشخص عند القدرة الآمنة وعدم تركه منفردًا في لحظة خطر واضح.
- إبعاد وسائل الأذى دون تعريض مقدم المساعدة للخطر.
- إشراك شخص بالغ موثوق أو ولي أمر عندما يتعلق الأمر بقاصر.
- عدم استخدام الموقع أو الاختبار الإلكتروني بديلًا عن التقييم العاجل.

## الأدوات والاختبارات

أي أداة تتعامل مع إيذاء النفس أو العنف أو الإساءة يجب أن:

- تكون غير تشخيصية ولا تنتج درجة خطر سريرية.
- تعرض الإجراء العاجل قبل التفسير عندما تشير الإجابة إلى خطر فوري.
- لا تقدم تعليمات دوائية أو ضمانات نتيجة.
- لا تؤخر المساعدة بإجبار المستخدم على التسجيل أو إكمال النموذج.
- توضح ما يحدث للبيانات، وتكون قابلة للاستخدام بلوحة المفاتيح وقارئات الشاشة والهاتف.
- تقدم مسارًا واضحًا بعد النتيجة.

## السجل والتصحيح

كل تعديل على خدمة أو رقم أو نطاق يسجل ما تغير، ولماذا، والمصدر الرسمي، وتاريخ التحقق، وهوية المراجع فقط إذا كان شخصًا حقيقيًا مخولًا. تزال الخدمة أو تتوقف عن النشر فور انتهاء الموعد أو ظهور خطأ أو تغير غير متحقق منه، مع حفظ الأثر التدقيقي الداخلي.

## حدود السياسة

هذه السياسة لا تنشئ شبكة طوارئ عالمية، ولا تثبت اعتماد أي جهة، ولا تقدم استشارة قانونية أو طبية. الخدمات والحقوق والأرقام تتغير حسب البلد والزمن، ولذلك تبقى بيانات تشغيلية تحتاج مراجعة دورية ومصدرًا رسميًا مباشرًا.
