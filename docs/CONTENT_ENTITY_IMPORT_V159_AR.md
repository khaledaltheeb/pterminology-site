# عقد استيراد كيانات المحتوى v159

## الهدف

يمنع هذا العقد أن تتحول زيادة عدد الصفحات إلى صفوف مجهولة الهوية أو بلا جمهور أو مستوى مخاطر أو مصادر. يبقى نموذج CSV القديم صالحًا مؤقتًا، لكن أي دفعة جديدة ينبغي أن تستخدم قالب v2.

## التوافق مع النموذج القديم

الملف `content/import-template.csv` يمثل v1. يفحصه المدقق ويقبل الصفوف الصحيحة مع تحذير ترحيل. لا يدعي هذا القبول أن الصف القديم يملك مراجعة أو مصادر أو تصنيف مخاطر.

## قالب v2

استخدم `content/import-template-v2.csv`. الحقول الإلزامية:

- `schema_version`: القيمة `2.0.0`.
- `entity_id`: معرف ثابت لا يتغير عند تعديل العنوان أو المسار.
- `slug`: مسار لاتيني ثابت بصيغة lowercase-kebab-case.
- `content_type`: نوع الكيان، مثل مصطلح أو اضطراب أو مقارنة أو دليل أو أداة.
- `ar` و`en`: الاسمان العربي والإنجليزي.
- `category`: التصنيف الموضوعي.
- `audience`: الجمهور الأساسي.
- `age_group`: الفئة العمرية.
- `risk_level`: منخفض أو متوسط أو مرتفع أو حرج.
- `description`: وصف عربي أصلي لا يقل عن 120 حرفًا.
- `keywords`: كلمات مرتبطة حقيقية مفصولة بـ`|`.
- `related`: slugs مرتبطة دون تكرار أو ربط ذاتي.
- `source_ids`: معرفات المصادر في عقد الأدلة، لا روابط عشوائية.
- `status`: مسودة أو مراجع أو منشور أو مؤرشف.
- `review_status`: حالة المراجعة الحقيقية دون ادعاء غير موثق.
- `reviewed_at`: تاريخ المراجعة عند انطباقه.

## مستويات المخاطر

### منخفض

محتوى واجهة أو تعريف غير سريري لا يقود إلى قرار صحي.

### متوسط

تثقيف نفسي عام وعلامات وأثر وظيفي. عند النشر يحتاج مصدرًا واحدًا على الأقل.

### مرتفع

التشخيص والتشخيص التفريقي والعلاج والتثقيف الدوائي وحماية الطفل وذوي الإعاقة. عند النشر يحتاج مصدرين على الأقل وحالة مراجعة مكتملة.

### حرج

إيذاء النفس والطوارئ وقرارات الدواء وادعاءات صلاحية الاختبارات. لا يسمح العقد بنشره من CSV إلا بحالة `externally-reviewed` وتاريخ مراجعة ومصادر محددة. هذا الشرط لا يثبت المراجعة بذاته؛ يجب أن يبقى الدليل محفوظًا وفق سجل الادعاءات المؤسسية.

## ما يمنعه المدقق

- تكرار `entity_id` أو `slug` بين الدفعات.
- عنوان عربي مفقود أو slug غير صالح.
- وصف رقيق أو كلمات مفتاحية غير كافية أو مكررة.
- ربط الصفحة بنفسها.
- معرفات مصادر غير صالحة أو مكررة.
- نشر محتوى متوسط أو مرتفع بلا الحد الأدنى من المصادر.
- نشر محتوى مرتفع بحالة مراجعة غير مكتملة.
- نشر محتوى حرج دون حالة مراجعة خارجية.
- رؤوس CSV جزئية أو غير معروفة.

## الحدود

المدقق يفحص بنية البيانات وعقود القبول. لا يثبت صحة النص العلمي، ولا يثبت أن المراجع اختصاصي حقيقي، ولا ينشر الصفوف تلقائيًا، ولا يحذف أو يضع noindex لأي صفحة. الربط الفعلي بالمولد يأتي بعد اعتماد العقد واختبار الترحيل على دفعة صغيرة.
