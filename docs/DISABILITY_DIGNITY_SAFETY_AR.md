# سياسة كرامة الأشخاص ذوي الإعاقة وسلامة المحتوى

**الحالة:** مسودة مؤسسية قابلة للمراجعة  
**آخر تحديث:** 2026-07-22  
**النطاق:** المحتوى المتعلق بالإعاقة، المتلازمات الوراثية، اضطرابات النمو العصبي، التوحد، اضطراب نقص الانتباه وفرط النشاط، صعوبات التعلم، واضطرابات التواصل والحركة والحواس.

## الغرض

تحدد هذه السياسة الحد الأدنى الملزم لكتابة ومراجعة ونشر المحتوى المتعلق بالأشخاص ذوي الإعاقة وأسرهم. هدفها حماية الكرامة، تقليل الوصم، منع الإكراه والاستغلال، ورفع وضوح الحدود العلمية والمهنية. لا تمثل هذه الوثيقة اعتمادًا طبيًا أو قانونيًا، ولا تدعي مراجعة اختصاصي ما لم يوثق ذلك صراحة.

## المبادئ الأساسية

1. الشخص ليس تشخيصًا ولا متلازمة. يجب ألا تختزل اللغة هويته أو قيمته أو مستقبله في حالة صحية أو نمائية.
2. تعرض الاحتياجات والتحديات إلى جانب نقاط القوة والقدرات والاختلافات الفردية، دون رومانسية زائفة أو إنكار للصعوبات الواقعية.
3. يمنع وصف الإنسان بأنه عبء أو مأساة أو عقوبة أو أقل قيمة، كما يمنع استخدام قصته لاستدرار الشفقة أو الترويج دون موافقة موثقة.
4. تستخدم لغة تحترم تفضيل الشخص والمجتمع المعني. عند اختلاف تفضيلات اللغة، يوضح الاختلاف دون فرض صيغة واحدة على الجميع.
5. لا يقدم المحتوى تشخيصًا ذاتيًا، ولا يعد بنتيجة علاجية، ولا يوصي بجرعات أو تعديل دواء أو إيقافه.

## الموافقة والمشاركة

- تراعى موافقة الشخص متى كان قادرًا على إعطائها، وتراعى مشاركته وقبوله العملي حتى عندما تكون الموافقة القانونية لولي أو ممثل.
- لا يجعل الامتثال الشكلي هدفًا علاجيًا بحد ذاته، ولا يبرر الإكراه أو التقييد أو العقاب أو الحرمان الحسي لمجرد تسهيل الإدارة.
- عند مناقشة دعم التواصل، يجب عرض وسائل التواصل البديل والمعزز واحترام الرفض والتوقف والإرهاق.
- لا تنشر صورة أو قصة أو تسجيلًا أو تفاصيل تعريفية لشخص ذي إعاقة دون موافقة موثقة ومحددة الغرض، مع مراعاة أن موافقة ولي الأمر لا تلغي كرامة الشخص وخصوصيته.

## الخصوصية ومنع الاستغلال

- يمنع طلب أو تخزين أسماء المرضى، أرقامهم، عناوينهم، سجلاتهم الطبية، صورهم، تفاصيل جلساتهم، أو بياناتهم المدرسية الحساسة داخل أدوات عامة أو قنوات بلاغ مفتوحة.
- يجب أن تشرح كل أداة ما تخزنه محليًا، وكيف يحذف، وما إذا كانت ترسل أي بيانات. الأصل هو عدم الإرسال إلى خادم دون موافقة موثقة ومحددة.
- يجب أن تتضمن الأدلة الأسرية إشارات واضحة إلى مخاطر التنمر والاستغلال المالي والجنسي والرقمي والإساءة والإهمال، وإلى ضرورة طلب حماية محلية مناسبة عند الاشتباه.
- لا تستخدم شهادات أو قصص نجاح أو صور قبل/بعد لإثبات فاعلية غير مدعومة أو لإيهام المستخدم بضمان النتيجة.

## اللغة والصور

### لغة يجب تجنبها

- أوصاف مهينة أو اختزالية أو ذات طابع شفقة أو رعب.
- تعميمات مثل: «جميعهم يفعلون» أو «لن يستطيع أبدًا» أو «عقل طفل» دون سياق علمي دقيق.
- وصف السلوك بأنه تلاعب أو عناد أو سوء تربية قبل استبعاد الألم والمرض والدواء والبيئة والحساسية والتواصل.
- استخدام مصطلحات تاريخية مهينة أو تصنيفات قديمة خارج سياق نقدي وتعليمي واضح.

### الصور والوسائط

- تمثل الأشخاص كأفراد كاملين في سياقات حياة طبيعية، لا كرموز للشفقة أو البطولة القسرية.
- يجب توفير نص بديل وظيفي، وتجنب كشف معلومات صحية غير لازمة.
- لا تستخدم صور تقييد أو انهيار أو أزمة كعنصر جذب بصري.

## المحتوى الوراثي والفحوص والخيارات الإنجابية

- تشرح المعلومات الوراثية والفحوص قبل الولادة وبعدها بلغة محايدة ودقيقة، مع توضيح حدود الاختبار واحتمالات الخطأ وعدم اليقين.
- يمنع توجيه الأسرة إلى قرار إنجابي بعينه أو الإيحاء بأن قيمة حياة الشخص تحددها نتيجة فحص.
- يجب فصل الحقائق الطبية عن القيم الشخصية والدينية والقانونية المتغيرة بحسب البلد.
- لا تقدم الاستشارة الوراثية أو القانونية بدل المختص، ولا تعرض قوانين أو حقوقًا محلية دون مصدر رسمي حديث ومحدد البلد.

## التدخلات والتعليم والدعم

- تميز بوضوح بين تدخلات مدعومة بأدلة، تدخلات واعدة محدودة الدليل، وتدخلات غير مدعومة أو قد تكون ضارة.
- يجب ذكر هدف التدخل، الفئة المناسبة، حدوده، آثاره المحتملة، وما إذا كان يقيس تحسنًا وظيفيًا ذا معنى للشخص لا مجرد زيادة الامتثال.
- يمنع تبرير الإهانة أو العزل أو التقييد أو الحرمان من التواصل أو الحركة أو الطعام كوسيلة تعليمية.
- يجب أن تكون مشاركة الشخص، كرامته، سلامته، وقدرته على التواصل معايير جودة أساسية.

## الأزمات والتغير المفاجئ

عند تغير السلوك أو القدرة بصورة مفاجئة، يجب أن يوجه المحتوى أولًا إلى استبعاد الألم والمرض والآثار الدوائية والتغيرات البيئية ومشكلات السمع والبصر والنوم والتواصل. تصبح الحاجة عاجلة عند وجود خطر وشيك على الشخص أو الآخرين، فقدان مفاجئ للمهارات أو الوعي، نوبة جديدة، إصابة، اشتباه إساءة أو استغلال، أو عجز مقدم الرعاية عن الحفاظ على السلامة. لا تستخدم المنصة بديلًا لخدمات الطوارئ أو الحماية المحلية.

## المراجعة العلمية والشفافية

- يجب توثيق الادعاءات الحساسة بمصادر مؤسسية أو أولية مناسبة، مع تاريخ مراجعة واضح.
- توصف حالة المراجعة بدقة: مسودة، تحتاج مراجعة، مراجعة لغويًا، مراجعة علميًا، أو منشورة. لا ترفع الحالة دون دليل.
- لا تنشأ أسماء خبراء أو لجان أو شراكات أو اعتمادات أو ادعاءات أثر.
- يجب الإفصاح عن تضارب المصالح والتمويل والشراكات الموثقة عند وجودها، وذكر عدم وجودها عندما يكون ذلك صحيحًا وقابلًا للتحقق.
- تصحح الأخطاء الجوهرية علنًا مع تاريخ التغيير ووصف موجز لما تغير، دون حذف سجل التصحيح لإخفاء الخطأ.

## قائمة قبول قبل النشر

- هل يحترم النص الشخص ولا يختزله في الحالة؟
- هل يوازن بين الاحتياجات ونقاط القوة دون إنكار أو مبالغة؟
- هل يشرح حدود التشخيص والعلاج والدواء؟
- هل استبعد الألم والمرض والبيئة والتواصل قبل تفسير السلوك؟
- هل يوضح الخصوصية والتخزين والحذف ومسار ما بعد النتيجة؟
- هل يحمي الموافقة والمشاركة والرفض والتواصل؟
- هل يتناول الاستغلال والإساءة والطوارئ عند الصلة؟
- هل المعلومات الوراثية والإنجابية محايدة وغير موجهة؟
- هل كل ادعاء حساس موثق وحالة المراجعة صادقة؟
- هل تخلو الصفحة من ادعاءات خبراء أو شراكات أو ضمانات غير موثقة؟

## ما لا تغطيه هذه السياسة

لا تستبدل هذه السياسة تقييمًا طبيًا أو نفسيًا أو تربويًا أو قانونيًا، ولا توحد القوانين والخدمات بين البلدان. عند غياب مصدر موثوق أو اختلاف التوصيات، يجب التصريح بعدم اليقين وتأجيل النشر بدل التخمين.
