# سياسة كرامة الأشخاص ذوي الإعاقة وسلامة المحتوى

**الحالة:** تحتاج مراجعة خارجية متخصصة  
**آخر تحديث:** 2026-07-22  
**النطاق:** الإعاقة، المتلازمات الوراثية، اضطرابات النمو العصبي، التوحد، اضطراب نقص الانتباه وفرط النشاط، صعوبات التعلم، واضطرابات التواصل والحركة والحواس.

## الغرض

تحدد هذه السياسة الحد الأدنى الملزم لكتابة ومراجعة ونشر المحتوى المتعلق بالأشخاص ذوي الإعاقة وأسرهم. هدفها حماية الكرامة وتقليل الوصم ومنع الإكراه والاستغلال ورفع وضوح الحدود العلمية والمهنية. لا تمثل اعتمادًا طبيًا أو قانونيًا، ولا تدعي مراجعة اختصاصي ما لم يوثق ذلك في سجل الادعاءات المؤسسية.

## المبادئ الأساسية

1. **الشخص ليس تشخيصًا.** لا تختزل هويته أو قيمته أو مستقبله في حالة صحية أو نمائية.
2. تعرض الاحتياجات والتحديات إلى جانب نقاط القوة والقدرات والاختلافات الفردية، دون رومانسية زائفة أو إنكار للصعوبات الواقعية.
3. يمنع وصف الإنسان بأنه عبء أو مأساة أو عقوبة أو أقل قيمة، كما يمنع استخدام قصته لاستدرار الشفقة أو التخويف أو الترويج.
4. تحترم اللغة تفضيل الشخص والمجتمع المعني، سواء فضّل لغة تضع الشخص أولًا أو لغة الهوية. لا تفرض صيغة واحدة على الجميع.
5. لا يقدم المحتوى تشخيصًا ذاتيًا، ولا يعد بالشفاء، ولا يوصي بجرعات أو تعديل دواء أو إيقافه.

## الموافقة والمشاركة والقرار

- تراعى موافقة الشخص متى كان قادرًا على إعطائها، وتراعى مشاركته وقبوله العملي حتى عندما تكون الموافقة القانونية لولي أو ممثل.
- يدعم المحتوى اتخاذ القرار بمساندة مناسبة بدل افتراض انعدام القدرة على الاختيار.
- لا يجعل الامتثال الشكلي هدفًا علاجيًا بحد ذاته، ولا يبرر الإكراه أو التقييد أو العقاب لتسهيل الإدارة.
- عند مناقشة التواصل، تعرض وسائل التواصل البديل والمعزز وتحترم إشارات الرفض والتوقف والإرهاق.
- لا تنشر صورة أو قصة أو تسجيلًا أو تفاصيل تعريفية دون موافقة موثقة ومحددة الغرض. موافقة ولي الأمر لا تلغي كرامة الشخص وخصوصيته.

## الخصوصية ومنع الاستغلال

- يمنع طلب أو تخزين أسماء المرضى وأرقامهم وعناوينهم وسجلاتهم الطبية وصورهم وتفاصيل جلساتهم أو بياناتهم المدرسية الحساسة داخل أدوات عامة أو قنوات بلاغ مفتوحة.
- تشرح كل أداة ما تخزنه محليًا، وكيف يحذف، وما إذا كانت ترسل بيانات. الأصل هو تقليل البيانات وعدم الإرسال إلى خادم دون موافقة محددة ومفهومة.
- تتضمن الأدلة الأسرية عند الصلة إشارات إلى التنمر والاستغلال المالي والجنسي والرقمي والإساءة والإهمال، ومسار طلب الحماية المحلية المناسبة.
- لا تستخدم قصص النجاح أو صور قبل/بعد لإثبات فاعلية غير مدعومة أو لإيهام المستخدم بضمان النتيجة.

## تفسير السلوك والتغير

لا يوصف السلوك بأنه تلاعب أو عناد أو سوء تربية قبل مراجعة الألم أو المرض، والآثار الدوائية، والنوم، والسمع والبصر، والبيئة الحسية، وعوائق التواصل، والتغيرات البيئية الحديثة. تأتي الفرضية النفسية أو السلوكية بعد هذه الاحتمالات الأساسية، لا بدلًا منها.

## اللغة والصور

### لغة يجب تجنبها

- الأوصاف المهينة أو الاختزالية أو ذات طابع الشفقة أو الرعب.
- تعميمات مثل «جميعهم يفعلون»، «لن يستطيع أبدًا»، أو «عقل طفل» دون سياق علمي دقيق.
- مصطلحات تاريخية مهينة خارج سياق نقدي وتعليمي واضح.
- ادعاءات التحويل إلى شخص «طبيعي» أو علاج الحالة بالكامل أو محو سمات الهوية.

### الصور والوسائط

- تمثل الأشخاص كأفراد كاملين في سياقات حياة عادية مع agency واختيارات، لا كرموز للشفقة أو البطولة القسرية.
- توفر نصًا بديلًا وظيفيًا وتتجنب كشف معلومات صحية غير لازمة.
- لا تستخدم صور التقييد أو الانهيار أو الأزمة كعنصر جذب بصري.
- تتطلب الوسائط التعريفية موافقة محددة قابلة للسحب.

## المحتوى الوراثي والفحوص والخيارات الإنجابية

- تشرح المعلومات الوراثية والفحوص قبل الولادة وبعدها بلغة محايدة ودقيقة، مع حدود الاختبار واحتمالات الخطأ وعدم اليقين.
- يمنع توجيه الأسرة إلى قرار إنجابي بعينه أو الإيحاء بأن قيمة حياة الشخص تحددها نتيجة فحص.
- تفصل الحقائق الطبية عن القيم الشخصية والدينية والقانونية المتغيرة حسب البلد.
- لا تقدم استشارة وراثية أو قانونية بدل المختص، ولا تعرض قانونًا أو حقًا محليًا دون مصدر رسمي حديث ومحدد البلد.

## التدخلات والتعليم والدعم

- تميز بوضوح بين تدخلات مدعومة بأدلة، وأخرى محدودة الدليل، وممارسات غير مدعومة أو قد تكون ضارة.
- تذكر هدف التدخل والفئة المناسبة وحدوده وآثاره المحتملة، وتقيس التحسن الوظيفي ذا المعنى للشخص لا مجرد زيادة الامتثال.
- يمنع تبرير الإهانة أو العزل الروتيني أو التقييد غير المبرر أو الحرمان من الطعام أو الماء أو التواصل أو الحركة كوسيلة تعليمية.
- تكون مشاركة الشخص وكرامته وسلامته وقدرته على التواصل معايير جودة أساسية.

## الأزمات والحماية

عند تغير السلوك أو القدرة بصورة مفاجئة، يوجه المحتوى أولًا إلى استبعاد الأسباب الأساسية المذكورة. تصبح الحاجة عاجلة عند خطر وشيك، أو اشتباه إساءة أو استغلال، أو نوبة جديدة أو فقد وعي، أو فقد مفاجئ للمهارات، أو إصابة مهمة، أو عجز مقدم الرعاية عن الحفاظ على السلامة. لا تستخدم المنصة بديلًا لخدمات الطوارئ أو الحماية المحلية.

## المراجعة العلمية والشفافية

- توثق الادعاءات الحساسة بمصادر مؤسسية أو أولية مناسبة مع تاريخ مراجعة واضح.
- توصف حالة المراجعة بدقة؛ لا ترفع إلى «مراجعة علميًا» أو «مراجعة من خبير» دون دليل وموافقة.
- لا تنشأ أسماء خبراء أو لجان أو شراكات أو اعتمادات أو ادعاءات أثر.
- يفصح عن تضارب المصالح والتمويل والشراكات الموثقة عند وجودها.
- تصحح الأخطاء الجوهرية علنًا مع تاريخ التغيير ووصف موجز لما تغير.

## قائمة قبول قبل النشر

- هل يحترم النص الشخص ولا يختزله في الحالة؟
- هل يوازن بين الاحتياجات ونقاط القوة دون إنكار أو مبالغة؟
- هل يشرح حدود التشخيص والعلاج والدواء؟
- هل يراجع الأسباب الأساسية قبل تفسير السلوك؟
- هل يحمي الموافقة والمشاركة والرفض والتواصل؟
- هل يوضح الخصوصية والتخزين والحذف وتدفق البيانات؟
- هل يتناول الاستغلال والإساءة والطوارئ عند الصلة؟
- هل المعلومات الوراثية والإنجابية محايدة وغير موجهة؟
- هل اللغة والصور قابلة للوصول وخالية من الاستغلال؟
- هل كل ادعاء حساس موثق وحالة المراجعة صادقة؟
- هل تخلو الصفحة من ادعاءات خبراء أو شراكات أو ضمانات غير موثقة؟

## ما لا تغطيه هذه السياسة

لا تستبدل هذه السياسة تقييمًا طبيًا أو نفسيًا أو تربويًا أو قانونيًا، ولا توحد القوانين والخدمات بين البلدان. عند غياب مصدر موثوق أو اختلاف التوصيات، يصرح بعدم اليقين ويؤجل النشر بدل التخمين.
