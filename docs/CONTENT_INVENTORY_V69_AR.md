# جرد جودة المحتوى v69

## الغرض

يفحص المدقق حزمة الموقع المولدة ويصدر تقريرًا لكل صفحة بصيغتي JSON وCSV. التقرير يساعد على ترتيب العمل، ولا يغير الصفحات أو خرائط الموقع أو تعليمات الفهرسة.

## القرارات

- `KEEP`: لا توجد إشارة آلية قوية تستوجب الإجراء.
- `IMPROVE`: الصفحة تحتاج تحسينًا في المحتوى أو البيانات الوصفية أو الروابط أو المصادر.
- `MERGE`: الصفحة نسخة مطابقة أو شديدة التشابه مع صفحة أقوى مرشحة لتكون الأصل.
- `NOINDEX`: صفحة خدمية أو صفحة تحمل تعليمات noindex بالفعل؛ القرار يحتاج مراجعة بشرية.
- `DELETE`: صفحة شبه فارغة ويتيمة؛ هذا اقتراح مراجعة فقط وليس أمر حذف.

## القياسات

- عدد الكلمات المرئية.
- العنوان والوصف وH1 وH2 وH3.
- Canonical وrobots وJSON-LD.
- الروابط الداخلية الصادرة والواردة والصفحات اليتيمة.
- نطاقات المصادر الخارجية مع استبعاد الشبكات الاجتماعية.
- التطابق النصي الكامل والتشابه العالي باستخدام shingles وJaccard.

## قواعد السلامة

1. لا حذف أو دمج أو noindex تلقائيًا.
2. القرارات الثلاثة تتطلب مراجعة بشرية ومراجعة أثر الروابط والزيارات والنية البحثية.
3. عدم وجود مصدر خارجي إشارة للتحسين، وليس حكمًا بأن المحتوى خاطئ.
4. عدد الكلمات وحده لا يثبت الجودة؛ يستخدم مع البنية والمصادر والروابط والتفرّد.
5. يجب إعادة تشغيل الجرد بعد كل دفعة إصلاح ومقارنة النتائج قبل/بعد.
