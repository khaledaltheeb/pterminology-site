# عقد تدقيق الأدوات التفاعلية v164

يبدأ هذا المسار Agent 06 فعليًا بمدقق مستقل يمر على صفحات HTML التفاعلية في حزمة الموقع ويخرج تقرير JSON وCSV قابلين للمراجعة.

## ما يفحصه

- وجود `lang` واتجاه RTL للصفحات العربية.
- وجود H1 واحد.
- تسمية حقول الإدخال والأزرار لقارئات الشاشة ولوحة المفاتيح.
- ظهور حد واضح بأن الأداة غير تشخيصية.
- إرشاد لطلب المساعدة المهنية.
- إعلان الخصوصية عند استخدام التخزين المحلي.
- لغة حذف وتصدير أو طباعة عند حفظ بيانات محلية.
- كشف استدعاءات إرسال شبكي مباشرة مثل `fetch` و`XMLHttpRequest` و`sendBeacon` و`WebSocket`.
- الصور التي تفتقد `alt` بوصفها تحذيرًا.

## الحدود

هذا المدقق لا يثبت الصلاحية السريرية للمقياس، ولا يثبت جودة التجربة الحية على المتصفح، ولا يغني عن Playwright وAxe وفحص الهاتف والحاسوب. دوره إنشاء جرد أولي قابل للمنع قبل ربطه بحزمة الإنتاج الموثقة.
