الثقة والسلامة والشفافية

توضح هذه الصفحة كيف تُبنى مواد المنصة، وما حدودها، وكيف نتعامل مع المصادر والمراجعات والخصوصية والتصحيحات والمواقف العاجلة.

المحتوى للتثقيف والدعم العام، وليس للتشخيص الذاتي أو وصف العلاج أو استبدال الطبيب أو المعالج. عند وجود خطر مباشر أو وشيك، تواصل فورًا مع خدمات الطوارئ المحلية أو أقرب جهة صحية عاجلة.

منهجية التحرير

نحدد سؤال الصفحة وجمهورها وحدودها، ثم نراجع المصادر الأولية والإرشادات المهنية، ونفصل الحقائق المدعومة عن التفسير العام والخطوات العملية. لا ننشر أسماء مراجعين أو خبراء ما لم تكن المراجعة موثقة فعليًا.

سياسة المصادر

نفضّل الإرشادات الرسمية، والمراجعات المنهجية، والدراسات المحكمة، والجهات الأكاديمية والمهنية المعروفة. لا تُعد الشهرة أو الترتيب في محرك البحث دليلًا على الجودة.

الخصوصية

الأدوات المحلية يفترض أن تعمل دون إرسال البيانات إلى خادم ما لم يُذكر خلاف ذلك بوضوح وتُطلب موافقة صريحة. لا ينبغي إدخال معلومات تعريفية حساسة في الحقول الحرة.

التصحيح والتحديث

يُسجّل تاريخ المراجعة وحالة المحتوى حيثما يدعمها النموذج. عند اكتشاف خطأ جوهري، تُصحح المعلومة ويُوثق التغيير بدل حذف أثره بصمت.

حالات المراجعة

  • مسودة: غير جاهزة للنشر.
  • تحتاج مراجعة: مكتملة بنيويًا لكن تحتاج تحققًا إضافيًا.
  • مراجعة تحريرية: فُحصت اللغة والبنية والمصادر الأساسية.
  • مراجعة علمية موثقة: لا تُستخدم هذه العبارة إلا عند وجود مراجعة فعلية موثقة من مختص مؤهل.

عدم ظهور اسم مراجع يعني أنه لا يجوز افتراض وجود مراجعة اختصاصية خارجية.

حدود الأدوات والاختبارات

أدوات المتابعة والفرز الذاتي تساعد على تنظيم الملاحظات أو فهم الأنماط، لكنها لا تثبت وجود اضطراب ولا تنفيه. يجب أن توضح كل أداة غرضها، وطريقة حسابها، وحدود النتيجة، والخصوصية، وما الخطوة التالية بعد النتيجة.

  • لا تُستخدم النتيجة وحدها لاتخاذ قرار دوائي أو إيقاف علاج.
  • الدرجات ليست تشخيصًا ولا حكمًا على الذكاء أو الشخصية.
  • أي تغير مفاجئ أو شديد يستدعي تقييمًا مهنيًا، خصوصًا عند وجود خطر أو ألم أو مرض محتمل.

البيانات المحلية والحذف

عندما تحفظ أداة بياناتها في المتصفح، يجب أن يكون الحفظ اختياريًا ومشروحًا بجوار الأداة. يمكن حذف البيانات من زر الحذف داخل الأداة عند توفره، أو من إعدادات المتصفح بحذف بيانات الموقع والتخزين المحلي لهذا النطاق.

حذف سجل المتصفح أو استخدام جهاز مشترك قد يؤدي إلى فقدان البيانات أو كشفها لمستخدم آخر؛ لذلك لا تستخدم التخزين المحلي للمعلومات شديدة الحساسية.

ملفات الارتباط والتقنيات المشابهة

لا نعرض حاليًا ادعاءً بوجود نظام تتبع أو تحليلات ما لم يكن مفعّلًا وموثقًا فعليًا. أي إضافة مستقبلية لملفات ارتباط غير ضرورية أو تحليلات خارجية يجب أن تُذكر بوضوح مع الغرض والمدة وخيار الرفض حيث يلزم.

سياسة التصحيح وسجل التحديثات

يمكن أن تشمل أسباب التصحيح: خطأ علمي، مصدر منتهي أو ضعيف، لغة وصمية، رابط مكسور، خلل في أداة، أو تنبيه سلامة ناقص. التصحيح الجوهري يجب أن يوضح ما تغيّر وتاريخ التغيير وحالة المراجعة الجديدة.

هوية الجهة والشراكات وتضارب المصالح

المنصة مشروع «مصطلحات علم النفس» المرتبط بالقنوات الرسمية الظاهرة في الموقع. لا تُفترض أي شراكة أو رعاية أو اعتماد مهني ما لم يُذكر اسم الجهة ونطاق العلاقة في صفحة عامة موثقة. لا توجد في هذه الصفحة أسماء خبراء أو جهات مراجعة غير مثبتة.

إذا وُجد مستقبلًا تمويل أو رعاية أو رابط تجاري يؤثر في مادة محددة، فيجب الإفصاح عنه بجوار المادة ذات الصلة، مع الفصل بين القرار التحريري والمصلحة التجارية.

الأزمات والمساعدة العاجلة

إذا كان هناك خطر مباشر أو وشيك على النفس أو الآخرين، أو فقدان وعي، أو تسمم، أو إصابة خطرة، أو عجز عن الحفاظ على السلامة، فلا تعتمد على الموقع أو أداة رقمية. اتصل بخدمات الطوارئ المحلية أو توجّه إلى أقرب قسم طوارئ، وابقَ مع الشخص إن كان ذلك آمنًا.

للمخاوف غير الوشيكة، تواصل مع طبيب أو أخصائي نفسي مرخص أو خدمة صحية موثوقة في بلدك. أرقام الطوارئ تختلف بين الدول، لذلك لا نعرض رقمًا عالميًا واحدًا قد يكون خاطئًا.

التواصل والإبلاغ

يمكن الإبلاغ عن خطأ أو رابط مكسور أو خطر سلامة عبر مستودع المشروع العام:

فتح بلاغ عام في GitHub Issues

تنبيه خصوصية: بلاغات GitHub عامة وقد تظهر في محركات البحث وتبقى محفوظة في سجل الخدمة. لا تكتب اسم المريض أو رقم الهاتف أو البريد أو العنوان أو التشخيص أو تفاصيل الجلسات أو الصور أو أي بيانات طبية أو تعريفية. استخدم وصفًا عامًا للمشكلة ورابط الصفحة فقط.

حدود قناة الإبلاغ: لا توجد حاليًا قناة سرية أو مشفرة لاستقبال البلاغات. لا تستخدم GitHub Issues لطلب مساعدة في أزمة، أو للإبلاغ عن إيذاء وشيك، أو لإرسال تفاصيل اعتداء أو استغلال، أو أي معلومات قد تكشف هوية شخص. في الحالات العاجلة استخدم خدمات الطوارئ أو الحماية أو الرعاية الصحية المحلية المناسبة.

الروابط الخارجية تنقلك إلى خدمات مستقلة تخضع لسياساتها وشروطها، وقد تسجل عنوان IP أو ملفات ارتباط أو بيانات استخدام وفق إعداداتها. وجود رابط لا يعني شراكة أو اعتمادًا، ولا تتحكم المنصة في احتفاظ الطرف الثالث بالبيانات.
