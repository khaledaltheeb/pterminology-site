## الإصدار السادس

- **200 صفحة مصطلح مستقلة** بالعربية والإنجليزية.
- **20 دليلًا عربيًا** مترابطًا مع صفحات المصطلحات.
- **15+ تصنيفًا موضوعيًا** بصفحات مستقلة.
- **240+ عنوان URL** داخل خريطة الموقع.
- بحث فوري بالعربية والإنجليزية دون إعادة تحميل الصفحة.
- مرشحات حسب التصنيف ومصطلح عشوائي ومفضلة محلية.
- PWA وService Worker والعمل الأساسي دون اتصال.
- RSS وOpenSearch وWeb App Manifest.
- API عام بصيغة JSON للمعجم وفهرس البحث.
- ملفات `robots.txt` و`sitemap.xml` و`llms.txt` و`security.txt`.
- ربط داخلي وBreadcrumbs وCanonical و`hreflang`.
- بيانات منظمة من Schema.org:
  - Organization
  - WebSite
  - CollectionPage
  - DefinedTerm
  - Article
  - BreadcrumbList

## ضمانات النشر

يبني GitHub Actions الموقع من مولّد Python متحقق منه، ثم يمنع النشر إذا لم تتوافر الشروط الأساسية، ومنها:

- 200 صفحة مصطلح بالضبط.
- 20 دليلًا بالضبط.
- 240 رابطًا على الأقل في Sitemap.
- وجود Canonical وJSON-LD و`hreflang`.
- وجود RSS وManifest وOpenSearch وAPI.
- عدم عودة أقسام الصفحة التي أُلغي اعتمادها.

بعد النشر تُرسل عناوين Sitemap إلى IndexNow، ويعمل تدقيق SEO أسبوعي للبيانات الوصفية والروابط الداخلية والتكرار.

## منهج المحتوى

المحتوى تثقيفي ومنظم لغويًا، مع تجنب الوصم والتشخيص السريع والادعاءات المطلقة. الهدف هو شرح المصطلح ووضعه في سياقه وربطه بالمفاهيم القريبة، لا اختزال الإنسان في تسمية واحدة.
