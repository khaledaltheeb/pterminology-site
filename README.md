# مصطلحات علم النفس | Psychology Terminology

موسوعة عربية منظمة تشرح مصطلحات علم النفس والصحة النفسية بلغة واضحة، وتربط المعرفة المكتوبة بالمحتوى المرئي المنشور عبر **@pterminology**.

## الروابط الرسمية

- المعجم: https://khaledaltheeb.github.io/pterminology-site/terms/
- المقالات: https://khaledaltheeb.github.io/pterminology-site/articles/
- Instagram: https://www.instagram.com/pterminology/
- YouTube: https://www.youtube.com/@psychology-term

- Sitemap: https://khaledaltheeb.github.io/pterminology-site/sitemap.xml

## بنية الموقع

- معجم عربي/إنجليزي للمصطلحات النفسية.
- ثلاثون دليلًا عربيًا قصيرًا مبنيًا على أسئلة البحث الشائعة.
- صفحات مستقلة بعناوين ووصف وCanonical وبيانات منظمة.
- ربط داخلي بين المعجم والمقالات والموضوعات ذات الصلة.
- Schema.org: Organization، WebSite، CollectionPage، DefinedTerm، Article وBreadcrumbList.
- بحث داخلي وOpenSearch وRSS وWeb App Manifest.
- خريطة موقع للصفحات والصور وملف Robots.
- إشعار IndexNow آلي بعد النشر.
- تدقيق أسبوعي تلقائي للروابط والبيانات الوصفية وخريطة الموقع.

## منهج المحتوى

المحتوى مكتوب للتثقيف العام، مع تجنب التشخيص السريع والوصم والادعاءات المطلقة. تُربط الصفحات بمصادر صحية رسمية عند الحاجة، ويُراجع المحتوى وفق الوضوح والدقة والفائدة للقارئ العربي.
